AGRI-TECH - Smart Agricultural Recommendation System

AGRI-TECH is an intelligent agricultural recommendation system that leverages machine learning to provide farmers with data-driven insights for optimal crop selection and fertilizer recommendations. Built with Flask and scikit-learn, this system analyzes soil conditions, weather parameters, and environmental factors to suggest the most suitable crops and fertilizers for maximum yield.

DocuQuery: Local Document Q&A with RAG

DocuQuery is a full-stack web application that allows you to ask questions about your own documents (.pdf, .docx, .txt) and get answers from a locally-run AI model. It uses a Retrieval-Augmented Generation (RAG) pipeline to ensure answers are based only on the content of your uploaded file, guaranteeing data privacy.
